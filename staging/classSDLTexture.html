Gedetailleerde Beschrijving

Manages an SDL_Texture loaded from a file.

SDLTexture encapsulates an SDL_Texture object, handling its creation from a file and destruction. It provides a method to access the underlying SDL_Texture pointer for use with SDL rendering operations. The texture is created given a renderer and a file path to an image resource.
