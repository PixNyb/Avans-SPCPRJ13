Detailed Description

Defines the BehaviourScript class as part of a game engine framework. BehaviourScript is a base class for scripting custom behaviors in game objects, providing a structure for event-driven programming in a gaming context. It offers a range of virtual methods for responding to common game events, such as start, update, and physics-triggered interactions.
