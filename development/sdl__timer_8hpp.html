The Timer class is used to create timing events within the game engine, such as counting down to a game event, tracking how long an operation takes, or managing in-game timers. It utilizes SDL's timing functions to provide accurate timing functionality.

This class is typically used to control game flow, animations, or time-dependent events. It can measure elapsed time or count down from a set interval.
