Gedetailleerde Beschrijving

The dependency injection container.

This container is used to store private or public instances for the game dev like SceneManager etc.

Noot
Refer to InstanceScope for more information about the scope of the instance.

Registers an instance.

Template Parameters
TThe type of the instance, this is what the instance is bound to. So you should use abstract base types here if there are any.
Parameters
instanceThe instance to register
scopeThe scope of the instance. Refer to InstanceScope for more information.
Waarschuwing
You can bind any type to any instance, but you should only bind abstract base types to instances.
