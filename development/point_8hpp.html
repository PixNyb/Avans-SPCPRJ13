Detailed Description

Defines the Point struct for representing 2D points and vectors in the DefunBobEngine. Point is a fundamental structure used throughout the engine to represent positions, directions, and other two-dimensional vector-like data. It can be used in various contexts such as graphics, physics, and user interface design.
