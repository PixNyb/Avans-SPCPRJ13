Detailed Description

Defines the Color class for representing RGBA colors in the DefunBobEngine. The Color class provides a representation of colors using the red, green, blue, and alpha (transparency) components. It also includes predefined static instances of common colors for convenience. Each color component is a double value ranging from 0 to 1.
