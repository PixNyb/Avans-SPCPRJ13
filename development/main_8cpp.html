Detailed Description

Entry point for the Engine, initializing and running the game loop.

Version
0.1
Date
07/11/2023

This file contains the main entry point of the Engine. It creates an instance of the GameLoop class and starts the game loop. The GameLoop class is responsible for managing the core game processes, including initializing the game state, handling input, updating game logic, and rendering frames. The 'run' method of the GameLoop kicks off the continuous loop that drives the game.
