The SDLWindow class provides a simple interface for creating, displaying, and managing an SDL_Window. It encapsulates SDL window creation and destruction and provides access to the underlying SDL_Window structure.

The Create function initializes the SDL_Window, and the Destroy function ensures it is properly destroyed when no longer needed.
