Detailed Description

Encapsulates the properties of a triangle and provides a drawing method.

SDLTriangle represents a simple geometric triangle defined by three vertices. It provides a method to render the triangle onto an SDL_Renderer, using the filledPolygonRGBA function from the SDL2_gfx library to draw a filled triangle.
