Detailed Description

Represents a piece of text in the user interface.

Warning
If you don't set width and height properly, text is not going to be shown.
Note
Height is boxed, width is wrapped. So if the text exceeds the width, it will be wrapped. If it exceeds the height, it will be boxed, so overflow will be invisible.

Text is a UIObject that manages the display of text, including its content, appearance, and position within the user interface.
