Gedetailleerde Beschrijving

A wrapper class for SDL_Thread to manage threading in the game engine.

This class abstracts the details of thread creation and control using SDL's threading capabilities. It provides methods to start, stop, and pause execution of threads and encapsulates the thread callback function within the class scope.
