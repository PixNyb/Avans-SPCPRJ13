The SDLTexture class is responsible for managing an SDL_Texture resource, providing an interface for creating and retrieving an SDL_Texture from a file path. It ensures that the texture is properly released when the object is destroyed. This class is typically used to handle the texture's lifecycle and to facilitate rendering with SDL2.
